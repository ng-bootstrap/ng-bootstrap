Navigate through options in the popup
ArrowUp Highlight above option
ArrowDown Highlight below option
mouseenter Highlight option under the mouse pointer

Note that starting keyboard navigation will always start from the currently highlighted option, no matter if highlighted from the keyboard or the mouse.

Select an option

Selecting means committing to the data model. It will display the currently selected option in the input. It will also make the popup close.

Enter or Tab Select highlighted option
click on option in the popup Select option under mouse pointer
Dismiss the popup
Escape Dismiss the popup, unless [autoClose] is false
click outside the widget Dismiss the popup, if [autoClose] is true or 'outside'
click inside the widget Dismiss the popup, if [autoClose] is true or 'inside'

If [autoClose] is false, you will have to use the API to dismiss the popup (either calling dismissPopup() on the instance or spawning an empty options list from the Observable).
