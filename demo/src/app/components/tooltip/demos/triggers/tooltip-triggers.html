You can easily override open and close triggers by specifying event names (separated by :) in the triggers property.

Note that mouseleave close trigger will not close the tooltip when mouse cursor is over it. Similarly, the focusout trigger will not close the tooltip when it is focused. Use the buttons below to see it in action.

Click the link to ng-bootstrap homepage. The tooltip is still open.

NB: These buttons use autoClose="outside" to enable clicks inside the tooltip.

Alternatively you can take full manual control over tooltip opening / closing events.
