You can easily override open and close triggers by specifying event names (separated by :) in the triggers property. You may pass multiple triggers; separate them with a space.

Two common options are mouseenter:mouseleave (alias hover) and focusin:focusout (alias focus).

Note that mouseleave close trigger will not close the popover when mouse cursor is over it. Similarly, the focusout trigger will not close the popover when it is focused. Use the buttons below to see it in action.

Click the link to ng-bootstrap homepage. The popover is still open.

NB: These buttons use autoClose="outside" to enable clicks inside the popover.
